Merhaba,

Bu yazımda kendi kullandığım ve sizler içinde faydalı olacağını düşündüğüm birkaç web sitesini paylaşacağım.

##  [Git Command Explorer](https://gitexplorer.com/) 
Bu interaktif site ile yapmak istediğiniz git işlemlerini seçiyorsunuz ve sizin için gerekli olan komutları hazırlayıp açıklamasını da yazıyor.

## [Flexbox playground](https://codepen.io/enxaneta/full/adLPwv)
CSS flexbox tüm yönleriyle yardımınıza yetişiyor ve anlık olarak parametrelerin değişimine göre sonuçlarını görebiliyorsunuz

## [Learn Git Branching](https://learngitbranching.js.org/)
İnteraktif bu site ile git komutlarını ve sonuçlarını görebilir ve kendi pratiklerinizi yaparak git konusunda gelişim sağlayabilirsiniz.

## [JSON ABC](https://novicelab.org/jsonabc/)
Bu site yardımı ile JSON içeriklerinizi A'dan Z'ye sıralayabilirsiniz.

## [JSON ABC and Diff](https://novicelab.org/jsonsortdiff/)
İki JSON içeriğinin sıralı olarak farkına bakabilirsiniz.

## [regular expressions 101](https://regex101.com/)
Online olarak regex hazırlayabilir ya da var olan regex tanımınızı test edebilirsiniz.

## [FAVICON GENERATOR](https://favicon.io/favicon-generator/)
Web siteniz için istediğiniz gibi favicon oluşturabilirsiniz.

## [My JSON Server](https://my-json-server.typicode.com/typicode/demo)
Projelerinizde test datalarına ve ortamına ihtiyacınız varsa bu site ile denemelerinizi kolayca yapabilirsiniz.

## [Online GUID / UUID Generator](https://www.guidgenerator.com/online-guid-generator.aspx)
Anlık ihtiyaçlar için GUID veya UUID ihtiyacınız varsa bu site yardımı ile istediğiniz koşula ve adet kadar kayıt oluşturabilirsiniz.

## [Interactive Vim tutorial](https://openvim.com/)
Vim konusunda kendiniz test etmek ya da pratik yapmak istiyorsanız bu interaktif site ile istediğiniz çalışmayı yapabilirsiniz.

## [StackBlitz](https://stackblitz.com/)
Online olarak hızlıca proje oluşturmanızı sağlayan ve içeriğinde birçok güncel framework için template bulunduran bir site.
